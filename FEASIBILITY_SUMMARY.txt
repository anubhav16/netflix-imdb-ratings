RECOMMENDED ARCHITECTURE
================================================================================

Data Flow:
  Detect thumbnail hover → Extract title → Check local cache
                                ↓
                        Cache hit? Return immediately
                                ↓
                        Cache miss? Queue for OMDb API
                                ↓
                        Batch fetch (rate-limited) → Store → Return
                                ↓
                        Inject overlay into DOM

Components:
  - content.js: DOM monitoring, hover detection, overlay injection
  - background.js: OMDb API calls, message routing, rate limiting
  - cache.js: IndexedDB wrapper for persistent caching
  - matchers.js: Title normalization, fuzzy matching
  - ui.js: Overlay creation and styling
  - styles.css: Non-intrusive overlay styling

Storage:
  - IndexedDB: Primary cache (large capacity, 30-day+ retention)
  - Chrome Storage API: Backup for user preferences

Rate Limiting:
  - OMDb free tier: 1,000 requests/day (0.7 req/sec sustained)
  - Queue requests to avoid exceeding limit
  - Alert user when approaching quota
